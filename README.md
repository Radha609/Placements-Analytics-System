# Placement Analytics System

## Overview
A SQL-based project for analyzing student placements, certifications, salary packages, and company-wise recruitment trends.

## Technologies Used
- MySQL
- SQL
- DB Fiddle

## Database Tables
- Students
- Certifications
- Companies
- Placements

## SQL Concepts Demonstrated
- Primary Keys
- Foreign Keys
- JOIN
- GROUP BY
- HAVING
- AVG()
- COUNT()
- MAX()
- ORDER BY

## Sample Analysis
- Average placement package
- Highest package offered
- Company-wise placement count
- Student certification analysis
